# Job Portal System

A job portal application built with PHP (procedural), HTML, CSS, JavaScript, and MySQL following MVC architecture.

## Features
- Company Profile Management
- Post Job
- Edit / Close Job
- View Applicants
- Shortlist / Reject Candidates

## Technologies Used
- PHP (Procedural)
- MySQL
- HTML/CSS
- JavaScript

## Setup Instructions
1. Import the database schema from `database/schema.sql`
2. Configure database settings in `config/database.php`
3. Run the application on a local server (XAMPP/WAMP)
